Должен содержать:
1) nick User1 - пользователь, содержащийся в тестовой бд изначально
2) nick User4 - пользователь, который должен быть создан в ходе тестирования
3) name Dial4 - диалог, который должен быть создан в ходе тестирования
4) id Dial1 - диалог, в котором состоит много пользователей вместе с User1
5) id Dial2 - диалог, в котором остался только User1
6) id Dial3 - диалог, в котором состоит много пользователей без User1
7) id M1 - сообщение в D1, в котором адресант User1, прочитано
8) id M2 - сообщение в D1, в котором адресант User2, не прочитано
9) id M3 - сообщение в D3, в котором User1 не адресант и не адресат, прочитано
10) id M4 - сообщение, которое должно быть отправлено User1 в Dial1 во время тестирования
11) id M5 - сообщение в D2, в котором адресант User1, прочитано.
